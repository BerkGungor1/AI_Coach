# AI_Coach

This repository contains large language models (LLMs) focused on text generation and question answering tasks.

Used LLM models are;
- Llama2 - 7B
- Falcon - 7B instruct
- Mistral - 7B instruct
- Zpehyr - 7B


Used datasets for fine-tuning;
- Ultrachat
- Baize
- Dolly-15k
- OpenAssistant Guanaco
